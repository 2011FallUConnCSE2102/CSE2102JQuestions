Save Question

In order to save the question you have created you MUST click either the 'Previous' or 'Next' buttons, even if there are no further questions. Once you do this, the current question and any changes you have made are automatically saved. This mechanism may seem slightly odd when you only have one question but it is particularly convenient when you are moving rapidly through a pool of questions making changes as you go.

Having saved the question you will still need to save the File. Before you do this however, you may want to set some configuration options for your set of questions.
