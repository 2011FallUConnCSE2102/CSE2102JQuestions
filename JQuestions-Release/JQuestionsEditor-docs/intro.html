What is JQuestions ?

JQuestions is an advanced exam simulator system for self-testing, learning and education. It is able to ask multi-choice and drag-and-drop type questions and has a variety of configuration features which make it ideal for revision, exam preparation, and general learning.

It has been deliberately written to be a small, simple to use, stand-alone piece of software. It depends on no other programs, database software or libraries being installed except for Java. It can be run as either a desktop application or as a Java Applet embedded within a browser. Despite it's small size, it offers a full range of configuration options and features, such as timers, embedding pictures in questions, optional immediate feedback to users, optional 'hints' may be set for users, results printing, answer-shuffling to prevent 'visually' remembering answers, and allowing up to 50 possible answers to each question.

All the code is written in standard Java 6. It is publicly available, open-source code which can be freely downloaded from the project's home page, http://sourceforge.net/projects/jquestions.

Who Is It For?

The JQuestions system is ideal for anyone who is studying, or who wants to learn a new subject or area. It is suited for self-learning, self-teaching and education. You can use it to create your own questions and tests on any subject, which you can keep and re-use at anytime. In this way you can build up your own library of tests and exams.

You can also swap JQS files with other people and JQuestions users. The definition of the JQS file format is open, publicly available and non-proprietary. This means that anyone else is also free to write their own applications programs to use and interact with JQS files, for whatever reason (e.g. to write a version for mobile devices, for instance), should they want to.

JQuestions, the JQuestions Editor, and the .JQS File Format

The JQuestions system consists of three elements:

  1. JQuestions.The exam simulator program which is highly configurable and can run in either 'Quiz mode' (where feedback, hints and explanations are available to the user), or in 'Exam mode' where no help or feedback is given. When it runs, the exam-simulator draws its questions in random order from the question pool.
  2. JQuestions Editor.The Editor program is used to create and edit questions and to manipulate JQS files.
  3. JQS File Format.The JQS file format is important because it is in the public domain and is an open format. This means other people are also free to write their own applications programs to use and interact with JQS files, for whatever reason (e.g. to write a version for mobile devices, for instance), should they choose to.

It is hoped that the openness of the JQS file format will enable the free sharing and exchange of information in a similar way to which the MP3 file format has facilitated the sharing and exchange of audio files.
