Other Menu Options

A few other simple menu options are available.

Options

File Open And Append

This menu option allows you to open a JQS file and append its questions to the current question pool. Using tihs option, it is possible to merge several files into one larger question pool.

File Save Marked Questions

This option allows you to save only questions which have been 'Marked' to a JQS file.

Delete Marked Questions

This option allows you to delete only questions which have been 'Marked' from the current question pool.

Help About

Display information about the JQuestions application.
