Configuration Options

A number of configuration options can be set for your pool of questions.

  • Description. Allows you to enter a brief text description of your pool of questions (e.g. '20 questions on mountaineering').
  • Allow User To Change These Configuration Settings. If 'Yes' then at exam-time the user will be able to change these configuration settings. If 'No' then at exam-time the user will not be given the option to change these configuration settings.
  • Mode. 'Exam Mode' - No help available to user at exam-time. No feedback given. No Hints or Explain texts available. 'Quiz Mode' (easier) - Help is available to user at exam-time. Feedback is given for each question (CORRECT or INCORRECT), Hints and Explain texts are available.
  • Timed. This may be timed or untimed. If 'timed' then a time limit must be set.
  • Number Of Questions. You do not have to ask all the questions in your pool. If your pool has say 100 questions you can use this option to specify that the user only has to answer a smaller number (say 20) questions. The 20 questions will then be randomly drawn from the pool of 100 questions. NOTE: zero means 'All' questions.
  • On Completion. This option allows you to specify what should happen when the user completes the exam.
