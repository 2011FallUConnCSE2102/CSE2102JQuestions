Create Questions

At present JQuestions is capable of handling two types of Question: multiple choice, or drag-and-drop.

Question Type

When you create a new question a dialog will appear which asks you to select which type of question do you want to create (see image). Make your selection, and click 'OK'. Once you have selected the appropriate blank question screen will be displayed.

Multi Choice Question

Below is the display screen for a Multiple Choice question with the main components labelled. Simply type the question text in the Main Question Panel. You may also choose menu options to allow you to 'Insert Image' into this panel. Optionally, you may add text to the Secondary Question Panel. Use the Spinner to set how many possible answers you want. Enter the text for each possible answer, and then click the checkbox for each correct answer.

Drag-And-Drop Question

Below is the display screen for a Drag And Drop question with the main components labelled. Simply type the question text in the Main Question Panel. You may also choose menu options to allow you to 'Insert Image' into this panel. Optionally, you may add text to the Secondary Question Panel. Use the Spinner to set how many possible answers you want. Enter the text for each possible answer, and then drag and drop the Possible Answer box into the appropriate location on the Main Panel. Note: the Possible Answer box will be dropped where the cursor is located, so you may want to position the cursor befoore dragging and dropping the box.
