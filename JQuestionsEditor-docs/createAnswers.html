Create Answers

Once you have created a question with a set of possible answers, you need to specify which are correct answers. (You must specify at least 1 correct answer).

Multi Choice Questions

To specify the correct answer(s) here, simply click the check boxes for the answers (e.g. 'A', 'B', 'C', ...) which are correct.

You can also specify some Hint text which may be of help to the user during the exam.

You can also specify an explanation of some kind for a particular possible answer by clicking its associated 'Explain' button. In the example, you may click 'Explain' for A (which is a correct answer) and enter some explanatory text (see image).

A combination of 'Explain' text for a question, as well as 'Explain' text for each individual possible answer can result in a useful detailed explanation in the JQuestions exam-simulator which may be displayed as below (see image).

Drag And Drop Questions

To specify the correct answers for a drag-and-drop type of question you need to create the possible answer boxes and then drag-and-drop each answer-box in its correct location on the Main Question Panel.

Use the spinner to set how many possible answers you want. Enter the text into each possible answer box. (Don't worry about the order possible answers are displayed since they will be randomly shuffled in the exam). Next, place the cursor at the correct location on the Main Question Panel. Finally, drag-and-drop the possible answer box to that location. Do this for each answer in the question.

Again, you have the option of setting Hint and Explain text for the whole question. In this example, we set no Hint text but we do enter some Explain text (see image). With a drag-and-drop question it is not possible to set detailed Explain text for each possible answer.
