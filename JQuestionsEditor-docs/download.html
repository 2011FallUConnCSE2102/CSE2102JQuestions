Download and Install

The JQuestions system has been deliberately designed to be small (< 1MB), easy to download and simple to install. It is freely available, like much open source software, from the Sourceforge network, at http://sourceforge.net/projects/jquestions.

Download

Once downloaded the files will need to be unzipped and installed into a directory or folder. The download should include a copy of the GNU Public License version 3.0, as well as a demonstration JQS file (demo-1.jqs) which can be used in both the JQuestions exam-simulator and Editor programs.

Installation

The programs should run on any system on which a Java Virtual Machine (JVM) is available. The install instuctions vary slightly depending on if you are installing on to a Microsoft Windows machine or a Linux, FreeBSD or Unix-type machine.

Installation: Microsoft Windows

On a Windows system JQuestions can be run by executing either of the two .bat files which come with JQuestions (JQuestions.bat, or JQuestionsEditor.bat). Simply open a command prompt window and type the command you want to start the program running. It is also possible to create an Icon on your desktop which 'links' to either of the batch files.

If you run the Editor program then you should see the main window (see image) appear.

Installation: Linux, FreeBSD or Unix-type System

If you are running a Linux, FreeBSD or Unix-type system then all you need to do is make sure a JVM is available and then open a command shell and execute the following command to run the Editor:

$ java -jar JQuestionsEditor.jar &

You should see the main window (see image) appear. Execute a similar command to get JQuestions up an running.
